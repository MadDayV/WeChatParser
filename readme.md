# Асинхронный Смарт-Парсер каталогов Szwego с интеграцией Generative AI

Профессиональный инструмент автоматизации сбора контента (текст, фабричные артикулы SKU, оригинальные медиафайлы) с платформы **Szwego** напрямую через API. Фреймворк поддерживает дуал-коннект к искусственному интеллекту (**Google Gemini** / **OpenAI** / **OpenRouter**) для автоматического перевода и SEO-оптимизации карточек товаров под e-commerce стандарты на русском языке.

## 🚀 Архитектурные преимущества (Security-First)

1. **Защита от антифрода WeChat**: Полный отказ от браузерной автоматизации (Playwright/Selenium). Скрипт работает на чистых асинхронных HTTP-запросах (`httpx`), имитируя заголовки легитимного Chromium-движка. Риск банов равен нулю.
2. **Тайм-курсорная пагинация (Time-Based Scroll)**: Автоматизирована логика листания каталога до любой глубины считыванием системной метки `update_time`.
3. **Двухэтапный конвейер**:
   * *Этап 1*: Медленный сбор данных из Китая с регулируемыми задержками (`SZWEGO_SAFE_PAUSE = 12.0`).
   * *Этап 2*: Пакетная генерация карточек через API ИИ на максимальной скорости с каскадной валидацией ID (`goods_id`).
4. **Изолированный промпт**: Системный промпт вынесен в текстовый файл `ai_prompt.txt`. Заказчик может редактировать ТЗ для ИИ напрямую через Блокнот, не затрагивая код программы.
5. **Защита от блокировки файлов**: Имя Excel-таблицы генерируется динамически с таймстампом (`products_export_YYYYMMDD_HHMMSS.xlsx`). Скрипт не упадёт, даже если предыдущий отчёт открыт в MS Excel.

## 🛠 Системные требования и установка

Для запуска требуется **Python версии 3.10 или выше**.

1. Распакуйте архив с проектом в рабочую директорию.
2. Установите зависимости одной командой:
   ```bash
   pip install -r requirements.txt
   ```

## ⚙️ Настройка конфигурации (`.env` и `ai_prompt.txt`)

### 1. Настройка параметров окружения в `.env`:
```env
ALBUM_ID=A201903291406004270013266   # ID альбома продавца из ссылки
TOTAL_TARGET_PRODUCTS=100            # Сколько товаров собрать за прогон
SZWEGO_SAFE_PAUSE=12.0               # Безопасная пауза в секундах
SZWEGO_TOKEN=Mzk4MDk3Q0E5RT...       # Токен авторизации (из Cookies, ключ 'token')
AI_PROVIDER=openrouter               # Провайдер: gemini / openai / openrouter / mock
OPENROUTER_API_KEY=sk-or-...        # Ключ OpenRouter (если выбран данный провайдер)
OPENROUTER_MODEL=google/gemini-2.5-flash # Используемая модель ИИ
```

### 2. Кастомизация промпта в `ai_prompt.txt`:
Файл находится в корне папки. Текст внутри используется моделью как `System Prompt`. Поддерживает HTML-разметку (`<h3>`, `<strong>`) для кастомизации описания.

## 📦 Запуск программы

Запуск осуществляется через интерактивное CLI-меню Windows-батника:
```bash
Запуск парсера.bat
```
*В меню доступны режимы массового парсинга, поштучного сбора по ссылкам, сборки итоговой таблицы и прямой вызов редактирования промпта в Блокноте.*
